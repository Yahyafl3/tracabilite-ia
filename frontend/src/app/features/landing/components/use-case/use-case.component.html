Cas d'usage principal : analyse de demandes de crédit

L'application analyse les informations financières d'une demande de crédit, génère une prédiction ML, explique les facteurs SHAP, consulte les agents Groq, calcule un consensus puis transmet le dossier à un validateur humain.
